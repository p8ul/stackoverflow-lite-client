## How should this be manually tested?
Clone repository
```
     $ git clone https://github.com/p8ul/stackoverflow-lite-client
     $ cd stackoverflow-lite-client
```

Install dependencies
```
      $ npm i
```
Start server
```
      npm start
```
Run test
 ```
      npm run test
```
